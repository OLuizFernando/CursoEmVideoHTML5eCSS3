Listas em HTML

Lista Ordenada

Código: 

<ol>
    <li> item 1 </li>
    <li> item 2 </li>
    <li> item 3 </li>
</ol>

Resultado:

  1. item 1
  2. item 2
  3. item 3

Parâmetro type=""

1 - Numerada (padrão)
A - Alfabética maiúsculo
a - Alfabética minúsculo
I - Algarismo romano maiúsculo
i - Algarismo romano minúsuculo

Parâmetro start=""

Indica o início da contagem

Lista Não Ordenada

Código: 

<ul>
    <li> item </li>
    <li> item </li>
    <li> item </li>
</ul>

Resultado:

  • item
  • item
  • item

Parêmtro type=""

disc - Bola preenchida (padrão)
circle - Bola não preenchida
square - Quadrado preenchido

Lista Com Sublista

Código: 

<ol>
    <li> item 1 </li>
    <li> item 2 </li>
    <ol type="a">
        <li> item 2.1 </li>
        <li> item 2.2 </li>
        <li> item 2.3 </li>
    </ol>
    <li> item 3 </li>
</ol>

Resultado:

  1. item 1
  2. item 2
    1. item 2.1
    2. item 2.2
    3. item 2.3
  3. item 3

Lista de Definições

Código: 

<dl>
    <dt> HTML </dt>
    <dd> Linguagem de marcação </dd>

    <dt> CSS </dt>
    <dd> Linguagem de estilização </dd>

    <dt> JavaScript </dt>
    <dd> Linguagem de programação </dd>
</dl>

Resultado:

HTML
Linguagem de marcação
CSS
Linguagem de estilização
JavaScript
Linguagem de programação
