Dicionário de tags HTML 📖

<h1> Define um Cabeçalho </h1>
<hr> Cria uma linha horizontal (horizontal row)
<p> Cria um novo parágrafo </p>
<br> Quebra a linha dentro do mesmo parágrafo (break row)

Para inserir emojis no site, usa-se o Codepoints do emoji eescolhido.
Por exemplo, o emoji de planeta usado no cabeçalho é definido pelo Codepoints "U+1F4D6 ".
Porém, no código, deve-se substituir o "U+" pelos caracteres "&#x".
