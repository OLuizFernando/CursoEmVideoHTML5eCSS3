Hyperlinks

href

Código: 

<a href="https://github.com/OLuizFernando">
    Visite o meu perfil no GitHub
</a>

Resultado:

Visite o meu perfil no GitHub

hreflang

Código: 

<a href="https://www.w3schools.com/html" hreflang="en">
    Site da W3Schools (em inglês)
</a>

Resultado:

Site da W3Schools (em inglês)

target

Código: 

<a href="https://github.com/OLuizFernando" target="_blank">
    Visite o meu perfil no GitHub (nova aba)
</a>

Resultado:

Visite o meu perfil no GitHub (nova aba)

Atributo target=""

_blank - abre o link em uma nova aba
_self - abre o link na mesma aba (padrão)

rel

Código: 

<a href="arquivos/pagina2.html" target="_self" rel="next">
    Próxima página
</a>

<br>

<a href="https://github.com/OLuizFernando" target="_blank" rel="external">
    Visite o meu perfil no GitHub (link externo)
</a>

Resultado:

Próxima página
Visite o meu perfil no GitHub (link externo)

Atributo rel=""

next - próxima parte
prev - parte anterior
author - site do mesmo autor
external - outro site
nofollow - site não endossado ou pago

download

Código: 

<a href="arquivos/teste.txt" download="teste.txt" type="text/txt">
    Download texte.txt
</a>

Resultado:

Download teste.txt

Atributo type=""

application/zip
text/html
text/css
text/javascript
video/mp4
video/H264
video/JPEG
audio/aac
audio/mpeg
font/ttf
image/jpeg
image/png
